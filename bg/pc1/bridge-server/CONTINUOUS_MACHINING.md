# 연속 가공 운영 메모

이 문서는 `bridge-server`의 **운영 참고 문서**입니다.

- 규칙 문서가 아닙니다.
- 최종 정책 기준은 루트 `rules.md`와 현재 코드입니다.
- 이 문서의 내용이 코드와 다르면 **코드가 우선**입니다.

## 1. 용도

- 연속 가공 관련 개념, 운영 점검 포인트, 로그 확인 키워드를 짧게 남깁니다.
- 상세 구현 규칙은 `rules.md`에 두지 않고 코드에서 확인합니다.

## 2. 운영 체크포인트

- 장비는 Idle 상태에서만 프로그램 교체를 시도합니다.
- 완료 감지는 장비 상태 변화와 생산 수량 변화를 함께 봅니다.
- 문제가 나면 브리지 서버 로그와 장비 상태를 먼저 확인합니다.

## 3. 문서 정리 원칙

- 과거 설계안, 오래된 슬롯 전략, 오래된 API 예시는 이 문서의 기준이 아닙니다.
- 현재 동작과 맞지 않는 내용은 누적하지 말고 삭제합니다.
